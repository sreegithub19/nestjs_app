 https://docs.nestjs.com/first-steps

 1. npm i -g @nestjs/cli
 2. nest new nestjs_app
 3. cd nestjs_app
 4. npm run start


 Deployment :
    1. link: https://trilon.io/blog/deploying-nestjs-to-zeit-now
    2. npm i -g now
    3. now login
    4. now.json (fill it)
    5. npm run build && now
    6. vercel --prod   ( to override later)
